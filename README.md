# BillingFlow API 💰

API de gestão de faturamento desenvolvida com .NET.

## 🚀 Tecnologias

- ASP.NET Core
- Entity Framework Core
- SQL Server
- JWT Authentication

## 📦 Funcionalidades

- Cadastro de clientes
- Criação de cobranças
- Registro de pagamentos
- Login com autenticação JWT

## 🔐 Autenticação

Utiliza JWT para proteger endpoints com [Authorize].
